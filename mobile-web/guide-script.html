.uiDropDown()

Default

피어나기 갑 천지는 그들에게 능히 것이다.
목숨을 구할 인생에 꽃이 끓는 희망의 창공에 길지 봄바람이다.
창공에 풀이 새 불어 있다.

Event

$('.js-ui-dropdown').on('uiDropDownOpened', function(){...});
레이어 열릴 때 발생
$('.js-ui-dropdown').on('uiDropDownClosed', function(){...});
레이어 닫힐 때 발생

Method

$('.js-ui-dropdown').uiDropDown('close');
레이어 닫기
$('.js-ui-dropdown').uiDropDown('open');
레이어 열기
$('.js-ui-dropdown').uiDropDown('update');
레이어 위치 업데이트
$('.js-ui-dropdown').uiDropDown('btnClose');
별도 버튼을 통해서 레이어 닫기 (접근성 포커스 관련 지원)
